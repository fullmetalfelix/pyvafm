def vafmcircuits.Machine.Assemble (   self)

Fabricator function.

This function is called when the machine is instantiated, only if the "assembly" parameter was given among initialisation arguments. The function is originally left unimplemented, so the user can build the setup after the machine is instantiated.

Example:

1 ...
2 # assembly function definition
3 def MyAssembly(compo):
4 
5  # add global channels
6  compo.AddInput("signal1")
7  ...
8  # add internal circuits
9  compo.AddCircuit(type='opAdd',name='adder',factors=2, pushed=True)
10  ...
11  # connect internal circuits to global input and outputs
12  compo.Connect("global.signal1","adder.in1")
13  compo.Connect("adder.out","global.out")
14  ...
15 
16 # main script
17 def main():
18 
19  main = Machine(name='machine', dt=0.01, pushed=True);
20 
21  # add a Machine circuit to main, and set it up with the MyAssembly function
22  main.AddCircuit(type='Machine', name='compo1', assembly=MyAssembly, pushed=True)
23  ...
24 
25 if __name__ == '__main__':
26  main()
