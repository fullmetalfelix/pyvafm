def vafmcircuits_Cantilever.InputQV (   self,
  args 
)

This is the circuit that allows calculates the cantilever oscilation, it contains both a vertial and lateral mode and can calculate a user defiend number of modes.

  • Initialisation parameters:
    • NumberOfModesV = Number of vertical modes
    • NumberOfModesL = Number of lateral modes
    • pushed = True|False
  • Input channels:
    • exciterz = The excitation of the vertical motion
    • excitery = The excitation of the lateral motion
    • record = This allows outputs to be controlled so the cantilever will only output when record = true
    • ForceV = This is the channel that must be connected to the z output of the interpolation circuit
    • ForceL = This is the channel that must be connected to the y output of the interpolation circuit
    • position = This is the channel that takes in the position of the cantilever holder and assigns holderx, holdery and holderz which is the acthul position of the cantilever in the forcefield.
  • Output channels:
    • zPos = The z displacment of the cantilever from the starting pos
    • yPos = The y displacment of the cantilever from the starting pos
    • xABSv = The displacment in the x axis of the vertical mode including the starting positon (starting position + holderx position)
    • yABSv = The displacment in the y axis of the vertical mode including the starting positon (starting position + holdery position + y tip position)
    • zABSv = The displacment in the z axis of the vertical mode including the starting positon (starting position + holdery position + z tip position)
    • xABSl = The displacment in the x axis of the lateral mode including the starting positon (starting position + holderx position)
    • yABSl = The displacment in the y axis of the lateral mode including the starting positon (starting position + holdery position + y tip position)
    • zABSl = The displacment in the z axis of the lateral mode including the starting positon (starting position + holdery position + z tip position)

Example:

1 canti = machine.AddCircuit(type='Cantilever', name= 'cantilever', NumberOfModesV=2, NumberOfModesL=0 ,pushed=True)
