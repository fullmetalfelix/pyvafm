488 #
489 # \image html waver.png "schema"
490 # Creates sine and cosine waves with the specifics given by the inputs.
491 # Can also create a sawtooth wave and a linear increasing signal (Ramper)
492 #
493 # - Initialisation parameters:\n
494 # - pushed = True|False push the output buffer immediately if True
495 #
496 # - Input channels:
497 # - \f$amp\f$ amplitude
498 # - \f$freq\f$ frequency
499 # - \f$offset\f$ offset value
500 # - \f$speed\f$ The rate at which the ramper will increase
