def vafmcore.VAFM.GetInternalChannel (   self,
  tag,
  chtype = ChannelType.Any 
)

Find the channel with the given tag among the subcircuits.

The tag of a channel is a string containing the name of the circuit to which it belongs and the name of the channel separated by a dot. The function raises an error if the circuit does not exist in the dictionary, or if the channel does not exist inside the circuit. The argument chtype is given as enum ChannelType, and if used it will limit the search for the channel only in the input or output channels of the circuit.

Parameters
tagChannel tag.
chtypeType of the channel: ChannelType.Input | ChannelType.Output | ChannelType.Any
Returns
Reference to the channel.

Example:

1 vafm.GetInternalChannel("oscillator.sin", vafmcore.ChannelType.Output)
2 vafm.GetInternalChannel("integrator.out", vafmcore.ChannelType.Output)
3 vafm.GetInternalChannel("mycircuit.mychannel", vafmcore.ChannelType.Any)
