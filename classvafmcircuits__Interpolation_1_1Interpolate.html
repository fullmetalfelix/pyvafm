Interpolate Class Reference

Detailed Description

This is the circuit that calculates the interpolation of a C-dimensional field in a D-dimensional space. The field must be in the following format: 

 x y z Fx Fy Fz
         or         
 x y z F
         or
 x Fx
         or
 x y Fx Fy

Initialisation parameters:

  • Filename = Filename of the force field input file.
  • Dimensions = Number of dimensons in the force field.
  • Components = Number of components of force in the force field.

Input channels:

  • coord : this is the coordiante to calcualte the interpolation.

Output channels: - Fx: The interpolated forces x component. - Fy: The interpolated forces y component. - Fz: The interpolated forces z component.

Example:

1 field = machine.AddCircuit(type='Interpolate',name='inter', Filename='forces.dat', Dimensons=3, Components=3 , pushed=True)
