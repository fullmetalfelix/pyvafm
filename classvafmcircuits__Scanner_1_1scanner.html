Detailed Description

This is the circuit that allows the user to control the cantilever placement.

  • Initialisation parameters:
    • process = Name of the object which was used to initiate the machine.
  • Input channels:
    • This circuit has no input channels.
  • Output channels: -Â pos: The position of the cantilever.
    • ScanPos: The positon of the cantilever when using the Scan function
    • Record: This is either True or False and works in conjunction with the cantilever output signal, this is used to control when you want the cantilever to output a signal, by default this is set to True

Example:

1 machine = Machine(machine=None, name='machine', dt=0.01, pushed=True);
2 scan = machine.AddCircuit(type='scanner',name='scann', Process = machine, pushed=True)
