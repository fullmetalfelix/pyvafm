vafmcircuits.Machine Class Reference

Detailed Description

Virtual Machine main class.

The vitual machine is effectively a Circuit, with input/output channels, but it also contains an internal assembly of circuits that run during the Update cycle. Every machine always has the output channel 'time' created during initialisation.
